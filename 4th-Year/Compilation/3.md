## Python Regex
- `r""`: raw strings to avoid escapes.
- `re.compile`.
	- `compiled.search(s)`.
- `d{4}`: capture exactly 4 digits.
- `P<var>` used to annotate regex to attach a name for later use.
- `compiled.finditer`: Returns an iterator of match objects.
