## Higher Order Functions
- HOF is a function which has an argument or a result which depends on a function.
```hs
f :: (a -> b) -> [a] -> [b]
map :: (a -> b) -> [a] -> [b]
map _ [] = []
mac succ [1,2,4] = [2,3,5] -- example
```
